Ecommerce-Backend Docs

This is an Ecommerce backend server which has a few endpoints, written in Typescript using node.js and express.js

  • Path: /signup Description: Signup and create a new user in the database, returns a token
  • Path: /login Description: Login to the system with the email and password
  • Path: /api/products Description: Get request gets all the products in json, Post request adds products to the Database
